======
MC2020
======

The '''MC2020''' generates 3 random bits whenever the input (the "clock") goes from low to high.

Sign parameters
===============

#. 3-BIT RANDOM
#. [MC2020]
#. Min Outputs:Max Outputs
#. Blank

Pins
====

Input
-----

#. Trigger IC

Output
------

#. Random Output
#. Random Output
#. Random Output
