======
MC3101
======

**MC3101** is an IC that decrementally counts down from X to 0 each time received input goes from low to high. When the counter reaches 0, it
outputs high.


Sign Lines
==========

1. DOWN COUNTER
2. [MC3101]
3. Start Value{:INF}
4. Current Value


IC Pins
=======


Inputs
------

- Decrement Counter
- Reset Counter
- Nothing

Outputs
-------

- High on Counter Complete
