============
Exit Remover
============

Exit Remover removes the boat when you exit it. If configured to, it will also add the boat item to your inventory.

Configuration
=============

========= ====================================== ======= =======
Node      Comment                                Type    Default 
========= ====================================== ======= =======
give-item Provide the player with the boat item. Boolean true    
========= ====================================== ======= =======
