======
MC6020
======

The '''MC6020''' generates 5 random bits whenever the input (the "clock") goes from low to high.

Sign parameters
===============

#. 5-BIT RANDOM
#. [MC6020]
#. Min Outputs:Max Outputs
#. Blank

====

-----

#. Trigger IC

Output
------

#. Random Output
#. Random Output
#. Random Output
#. Random Output
#. Random Output
