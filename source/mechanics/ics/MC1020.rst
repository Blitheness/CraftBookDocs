======
MC1020
======

The **MC1020** generates a random state whenever the input (the "clock") goes from low to high.

Sign parameters
===============

   1. Blank
   2. [MC1110]
   3. Min Outputs:Max Outputs
   4. Blank


Sign Lines
==========

1. RANDOM BIT
2. [MC1020]
3. Min On:Max On
4. Blank


IC Pins
=======


Inputs
~~~~~~

- High to output random state

Outputs
~~~~~~~

- Outputs random state
